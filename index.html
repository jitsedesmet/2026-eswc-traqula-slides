Plug-and-play grammar rules

Rule Dictionary

iriFull <…>

Builder Operations
+
.addRule(…) Register a new rule module
~
.patchRule(…) Replace a rule's behavior
−
.removeRule(…) Drop a rule (restrict language)
TypeScript Example
const base = GeneratorBuilder
  .create([iriFull, prefixedIri])
  .addRule(iri)
  // Base IRI generator

const custom = GeneratorBuilder
  .create(base)
  .patchRule(altPrefixedRule)

What is preserved vs. what changes:
Preserved (source-identical): keyword casing SelECT, extra spacing ?p   ?o, comments, punctuation
Changed (intentional): variable name ?s → ?subject — only the patched AST node is regenerated
